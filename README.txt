Description:

This project is a small and simple example for getting and validating the responses from oxford dictionary api
for various scenarios on different end points 

End points covered for oxford API:

1. Retrieve the origin of words
2. Validate scenarios for Entries Get call with 400 and 404
3. Retrieve translation for a particular text in any given language
4. Validate Translation endpoint for 400 and 404

Implementation details:

Used Rest assured using Java 8 precisely, Cucumber, Log 4j and JUnit

Usage:

Run using Maven command - mvn test

Report:
test-output.html can be found under root folder and can be viewed in browser
